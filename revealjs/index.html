RevealJS

Schicke Präsentationen im HTML-Format

Beschreibung

Bei RevealJS handelt es sich um ein Open Source Präsentationstool. Gestaltet werden die Präsentationen im HTML-Format. Auf diese Weise können sie einfach weiter genutzt werden. Es gibt eine nette Features wie z.B. einen Gif-Hintergrund und unterschiedliche Animationseffekte.

Empfehlenswert ist eine Veröffentlichung über Github Pages. Über diesen Weg können die Präsentationen sowohl als HTML geteilt, als auch direkt online präsentiert werden. Wenn dieser Weg einmal bekannt ist, dann eignet er sich auch darüber hinaus für die Veröffentlichung von OER in unterschiedlichen Formaten.

Kennenlernen und ausprobieren

Ein ausführliches Beispiel zum Ansehen gibt es mit dem Slide Verzeichnis des eBildungslabor. Mit der dort veröffentlichten Schritt für Schritt Anleitung lässt sich eine erste Präsentation remixen.

Wer eine einfachere Variante (= kein Code nötig) zur Erstellung und zum Remix von Präsentationen sucht, kann zudem Slidewiki ausprobieren.

Weiterführende Links
