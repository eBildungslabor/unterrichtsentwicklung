H5P

Interaktive Online-Selbstlernübungen und mehr

Beschreibung

H5P ist eine sehr vielfältig nutzbare Open Source Software mit dem Fokus der Erstellung interaktiver Online-Lernmaterialien. Das Tool umfasst mehrere so genannte Inhaltstypen. Dazu zählen unter anderem interaktive Videos, Quizze, Karteikarten oder Drag and Drop Aufgaben. Damit sind die Möglichkeiten von H5P aber noch nicht erschöpft. Das Tool lässt sich z.B. auch für Projektplanung oder für Präsentationen nutzen.

H5P Materialien werden im Format .h5p erstellt. Dieses Format kann online genutzt, beliebig auf andere Websites eingebettet und mithilfe eines Editors erstellt und remixt werden. Der Editor ist als Plugin für Drupal, Moodle, Ilias und Wordpress verfügbar.

Kennenlernen und ausprobieren

Auf der Website EinstiegH5P.de kannst Du erste H5P-Inhalte ohne Registrierung testweise erstellen. Für eine dauerhafte Speicherung muss ein kostenfreier Account angefragt werden. Zum Starten eignen sich simple Inhaltstypen am besten - z.B. Multiple Choice.
