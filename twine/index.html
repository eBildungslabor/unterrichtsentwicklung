Twine

Interaktive Geschichten, Spiele und Tutorials

Beschreibung

Twine ist ein Open Source Tool zur Erstellung interaktiver Texte. Vor diesem Hintergrund kann es genutzt werden für Geschichten mit mehreren Handlungssträngen, für Tutorials oder für Spiele.

Auf der Website Twinery werden erstellte Twines nur lokal gespeichert. Zur Weiternutzung können sie als HTML-Datei exportiert und als solche beliebig im Internet veröffentlicht werden. Eine explizites Angebot hierfür ist Philome.la
